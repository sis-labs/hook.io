Route Formatting

:name a named parameter to capture from the route
*splat a wildcard splat to capture from the route
() Optional group that doesn't have to be part of the query. Can contain nested optional groups, params, and splats
strings Anything else. Interpolate static strings with params.

Route Examples

/:id/create
/some/(optional/):thing
/users/:id/comments/:comment/rating/:rating
/*a/foo/*b
/books/*section/:title

Service Security

 
In order to enable private services, you must upgrade to a paid account.

A public service will be publicly indexed and available for anyone to access at anytime.

A private service will be hidden and only allowed accessible through a role based key access credential.

Services can be scheduled on a timer

All Services support the ability to schedule their execution based on a date time pattern.

We call this date time pattern a Cron.

By design, Crons will not pass any data into the Hook. If you need to pass data into the scheduled Hook, you can simply set default data in the Hook or chain two Hooks together to pipe the data.

Warning: We currently cannot provide any guarantees about Cron accuracy or reliability. If you require a high reliability timer, please let us know.
