Getting Started

The {{appName}} client SDK makes it easy to interact with the {{appName}} platform from any service or third-party application.

Remember, you don't have to use the {{appName}} Developer API from an outside system! {{appName}} provides free hosting for Microservices and Webhooks. These hosted services have full integrated access to the {{appName}} Developer API. All JavaScript based Hook services have an available Hook.sdk parameter which contains the SDK.

If you intend to run a block of code in response to HTTP requests or as a scheduled task we recommend Creating a new hosted Microservice.
